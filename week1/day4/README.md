# 🔄 Types of Flows Learned

## Record-Triggered Flow
Automatically runs when:
- A record is created
- A record is updated
- A record is deleted

---

## Screen Flow
Used to:
- Collect user input
- Build guided forms
- Create interactive automation

---

## Autolaunched Flow
Runs automatically in the background without screens.

---

# 🧠 Flow Builder Components

Learned important Flow Builder elements:

## Start Element
Defines:
- Trigger object
- Trigger conditions
- When the flow runs

---

## Get Records
Used to:
- Retrieve Salesforce records
- Query related data

---

## Update Records
Used to:
- Modify existing records automatically

---

## Create Records
Creates new Salesforce records automatically.

---

## Decision Element
Used for:
- Conditional logic
- Multiple flow paths

---

## Action Element
Used to:
- Send emails
- Post to Chatter
- Submit approvals
- Trigger platform actions
